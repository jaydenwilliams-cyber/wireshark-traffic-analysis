# Wireshark Traffic Analysis Automation

## Overview
This project captures network traffic using Wireshark and analyzes packet data using Python. The script automates identification of top IPs, protocols, and port activity.

## Tools Used
- Wireshark
- Python
- Pandas

## Features
- Captures live network traffic
- Processes packet data from CSV
- Identifies top source IPs
- Identifies top destination IPs
- Summarizes protocol usage
- Extracts port activity
